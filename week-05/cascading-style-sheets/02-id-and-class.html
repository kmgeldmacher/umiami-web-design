id and class

id and class attributes allow us to further specify which elements we want to apply styles to.

id

id can be applied once

  • appears in an element as id="value"
    • Example: <p id="first"></p>
  • appears in CSS prefaced by the pound sign or hashmark (#)
    • Example: #first { color: blue; }

class

class can be applied multiple times

  • appears in an element as class="value"
    • Example: <p class="blue">
  • appears in CSS prefaced by a period (.)
    • Example: .blue { color: blue; }
