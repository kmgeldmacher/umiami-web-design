CSS Anatomy

This is a basic CSS block:

selector { property: value; }

The selector is the element in HTML we want to style. This can be a specific element such as p, h1or strong, an id or class.

The portion enclosed within the curly braces ({ }) is called the declaration. We can specify multuple declarations for each selector. There are hundreds of properties we can change.

Properties and values are separated by a colon (:). All declarations should have a semicolon (;) at the end.
