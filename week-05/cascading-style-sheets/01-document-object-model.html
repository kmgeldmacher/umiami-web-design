Document Object Model

We might describe the DOM as a family tree or tree-like structure for HTML.

Elements nested inside elements are 'children.'

Elements that wrap elements are 'parents.'

Inspecting the DOM

Google Chrome's Dev Tools has a built-in DOM inspector.

There are three ways to open the inspector:

  • View > Developer > Developer Tools
  • Click on the Hamburger Menu in the top right corner > Tools > Developer Tools
  • Command-Option-C

Once Dev Tools is open, select Elements

Using Dev Tools

Here's the thing about Dev Tools: They are fantastic to help you understand the code for web sites, but they are also very complex. Trying to learn everything about Dev Tools is probably akin to learning the vast array of features Adobe Creative Suite has.

You can edit your HTML and CSS in Dev Tools, but those changes will not be saved. If you refresh the page, all changes will be lost.

Think of Dev Tools as a feature that can help you learn what the different HTML elements do, how different CSS properties display and as a debugging tool. As you become more comfortable with web basics, you may want to learn more about how you can intergrate Dev Tools into your workflow.
