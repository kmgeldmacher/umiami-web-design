Exercise: Apply a Style Sheet!

Remember all those questions you asked about making your pages look good? Now you have the tools to do so!

Take the Presidential Election page you have been working on and apply an external style sheet to it. (Make sure you place your stylesheet in the css directory within your exercise's directory. Keep your files organized!)

Try to change properties related to alignment, fonts, sizes, background colors (maybe even background images!), colors, borders, list styles, opacity, how an item displays (block, inline -- can you change that?) etc. WebPlatform has a comprehensive list of CSS properties -- there are 346 of them! -- and the values they accept.

If you need to modify your HTML, that is okay.

Go crazy! Well, kind of. Remember, this is a design class. Don't turn your page into a mess by committing design sins (i.e. poor font combinations, misaligned elements, poor color or background choices)!

Assignment for Wednesday

Come to class with several properties (and the values they accept) you learned about and what they do. Or perhaps you discovered something about the way CSS can be applied? (Yes, that's kind of a hint.)

Here's the catch: Everyone has to present something different. No duplicate information. I will draw names at random. Anyone who cannot offer a unique property will have one half of the day's class participation points deducted from their final grade.

Important Note

I highly recommend avoiding properties that involve margins, padding, positioning, etc. These are related to the box model and layout, which we will go over in a couple weeks. These aspects of CSS can be tricky, so better not to confuse and frustrate yourself right now.

Tips for debugging CSS

The W3C has a CSS Validator. If you find a style isn't applying as you might expect it to, run it through the validator to see if you have any syntax errors.

If a property you have applied still does not do anything, it might be related to one of the areas I suggested you avoid. Or it may need another property set to work correctly.
